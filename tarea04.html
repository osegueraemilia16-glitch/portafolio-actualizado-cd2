CSS3 es el lenguaje que transforma un documento HTML sin estilo en una página visualmente atractiva. Esta guía te llevará desde cero hasta dominar los conceptos fundamentales: selectores, el modelo de caja, colores, tipografía, Flexbox, Grid, animaciones y más. Capítulo 01

¿Qué es CSS3?

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje que controla la presentación visual de las páginas web: colores, tipografías, márgenes, tamaños, animaciones y el diseño en pantalla.

CSS3 es la versión actual, introducida gradualmente desde 2011. Trajo módulos independientes como Flexbox, Grid, variables CSS, animaciones, filtros y media queries que revolucionaron el diseño web.

🎨

Presentación visual

CSS controla todo lo que ves: colores, fuentes, tamaños, bordes, fondos y espaciados.

📱

Responsivo

Con media queries, una misma hoja de estilos adapta el diseño a celulares, tabletas y computadoras.

✂️

Separación de responsabilidades

HTML define la estructura; CSS define la apariencia. Mantenerlos separados hace el código más limpio y fácil de mantener.

🔁

Cascada y herencia

Los estilos se "cascadean": un elemento hijo hereda propiedades de su padre y los estilos más específicos tienen mayor prioridad.

Las 3 formas de escribir CSS:

formas-de-css.html
<!-- 1. CSS en línea (solo casos excepcionales) -->
<p style="color: blue; font-size: 18px;">Texto</p>

<!-- 2. CSS interno (en el <head>) -->
<style>
  p { color: blue; }
</style>

<!-- 3. CSS externo (RECOMENDADO) -->
<link rel="stylesheet"
      href="estilos.css"/>
💡

Punto clave: Siempre usa un archivo CSS externo (.css). Así un solo archivo de estilos puede aplicarse a todas las páginas de tu sitio, y cambiar el diseño de todo el sitio es tan simple como editar un solo archivo.

Capítulo 02

Selectores: cómo apuntar a los elementos

Un selector es la parte del CSS que indica a qué elemento HTML se aplicarán los estilos. Dominar los selectores es la habilidad más importante en CSS.

selectores.css
/* ── Por etiqueta (todos los párrafos) ── */
p { color: #444; }

/* ── Por clase (elementos con class="tarjeta") ── */
.tarjeta { border-radius: 12px; }

/* ── Por ID (elemento único con id="menu") ── */
#menu { background: #1572b6; }

/* ── Descendiente (p dentro de .contenedor) ── */
.contenedor p { font-size: 0.9rem; }

/* ── Hijo directo ── */
nav > a { text-decoration: none; }

/* ── Pseudo-clase: estado hover ── */
a:hover { color: #e91e8c; }

/* ── Pseudo-clase: primer hijo ── */
li:first-child { font-weight: bold; }

/* ── Pseudo-elemento: primera letra ── */
p::first-letter { font-size: 2rem; color: #1572b6; }

/* ── Selector múltiple ── */
h1, h2, h3 { font-family: 'Playfair Display', serif; }

/* ── Selector universal ── */
* { box-sizing: border-box; }

Punto clave – Especificidad: Cuando dos reglas CSS apuntan al mismo elemento, gana la de mayor especificidad. Orden: !important > estilos en línea > ID > clase > etiqueta. Evita usar !important siempre que puedas.

Capítulo 03

El modelo de caja (Box Model)

Todo elemento HTML es una caja rectangular. El modelo de caja define cómo se calcula el tamaño total de esa caja con cuatro capas.

box-model.css
.caja {
  /* Tamaño del contenido */
  width:  300px;
  height: 200px;

  /* Relleno interior (entre contenido y borde) */
  padding: 20px;            /* todos los lados */
  padding: 10px 20px;      /* arriba/abajo  izq/der */
  padding-top: 10px;       /* lado individual */

  /* Borde */
  border: 2px solid #1572b6;
  border-radius: 12px;      /* esquinas redondeadas */

  /* Margen exterior (entre la caja y otros elementos) */
  margin: 16px;
  margin: 0 auto;           /* centrar horizontalmente */

  /* IMPORTANTE: incluir padding en el width total */
  box-sizing: border-box;
}
📦

Content

El área donde va el texto o las imágenes. Se controla con width y height.

🛋️

Padding

Espacio interior entre el contenido y el borde. Tiene el color de fondo del elemento.

🖼️

Border

El marco que rodea el padding. Puede ser sólido, punteado, con color y grosor.

↔️

Margin

Espacio exterior entre la caja y los demás elementos. Es siempre transparente.

🎯

Punto clave – box-sizing: Agrega siempre * { box-sizing: border-box; } al inicio de tu CSS. Así el padding y el borde se incluyen dentro del width declarado, evitando desbordamientos inesperados.

Capítulo 04

Colores, fondos y gradientes

CSS3 ofrece múltiples formatos para expresar colores y herramientas poderosas para crear fondos atractivos.

colores-fondos.css
.elemento {
  /* Formatos de color */
  color: red;                       /* nombre */
  color: #e91e8c;                   /* hexadecimal */
  color: rgb(233, 30, 140);         /* RGB */
  color: rgba(233, 30, 140, 0.5);   /* RGB + opacidad */
  color: hsl(328, 81%, 52%);        /* HSL */

  /* Color de fondo sólido */
  background-color: #fce4ec;

  /* Imagen de fondo */
  background-image: url("fondo.jpg");
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;

  /* Degradado lineal */
  background: linear-gradient(
    135deg, #1572b6, #e91e8c
  );

  /* Degradado radial */
  background: radial-gradient(
    circle, #33a9dc, #1572b6
  );

  /* Opacidad del elemento completo */
  opacity: 0.8;
}

Capítulo 05

Tipografía y texto

CSS ofrece control total sobre la apariencia del texto: fuente, tamaño, peso, espaciado, alineación y más.

tipografia.css
/* Importar fuente de Google Fonts */
@import url('https://fonts.googleapis.com/css2?
  family=Playfair+Display:wght@700&
  family=Lato:wght@300;400;700&display=swap');

body {
  font-family: 'Lato', sans-serif; /* fuente + respaldo */
  font-size:   16px;              /* tamaño base */
  line-height: 1.6;               /* interlineado */
  color:       #1a1a1a;
}

h1 {
  font-family:    'Playfair Display', serif;
  font-size:      2.5rem;      /* relativo al tamaño base */
  font-weight:    700;          /* 100 a 900 */
  letter-spacing: 0.05em;      /* espaciado entre letras */
  text-transform: uppercase;   /* MAYÚSCULAS */
  text-align:     center;
}

p {
  font-size:       1rem;
  text-indent:     1.5em;       /* sangría primera línea */
  word-spacing:    2px;
  text-decoration: none;        /* underline, line-through */
}

Capítulo 06

Display y posicionamiento

La propiedad display y las propiedades de posicionamiento controlan cómo y dónde aparece cada elemento en la página.

display-position.css
/* DISPLAY: tipo de renderizado */
.bloque    { display: block; }        /* ocupa toda la línea */
.en-linea  { display: inline; }       /* solo el ancho del contenido */
.oculto    { display: none; }         /* invisible y sin espacio */
.invisible { visibility: hidden; }   /* invisible pero conserva espacio */

/* POSITION: tipo de posicionamiento */
.estatico  { position: static; }     /* por defecto, flujo normal */
.relativo  {
  position: relative;
  top: 10px; left: 20px;            /* desplazado de su lugar original */
}
.absoluto  {
  position: absolute;
  top: 0; right: 0;                 /* respecto al padre relativo */
}
.fijo      {
  position: fixed;
  bottom: 20px; right: 20px;        /* fijo en la pantalla al hacer scroll */
}
.sticky    {
  position: sticky;
  top: 0;                           /* se pega al borde al hacer scroll */
}
.encima    { z-index: 10; }           /* orden de apilamiento */

Capítulo 07

Flexbox: diseño unidimensional

Flexbox es el sistema de diseño de CSS3 para organizar elementos en una fila o columna con control total sobre alineación y distribución del espacio.

flexbox.css
/* Contenedor flex */
.contenedor {
  display:         flex;
  flex-direction:  row;            /* row | column | row-reverse */
  justify-content: space-between; /* alineación eje principal */
  align-items:     center;         /* alineación eje cruzado */
  flex-wrap:       wrap;           /* permitir salto de línea */
  gap:             16px;           /* espacio entre hijos */
}

/* Hijos flex */
.hijo {
  flex: 1;           /* crecer y ocupar espacio disponible */
  flex-grow:   1;    /* cuánto puede crecer */
  flex-shrink: 0;    /* si puede encogerse */
  flex-basis: 200px; /* tamaño base antes de crecer */
  align-self: flex-end; /* alineación individual */
}

/* justify-content: valores clave */
/* flex-start | flex-end | center  */
/* space-between | space-around | space-evenly */

Capítulo 08

CSS Grid: diseño bidimensional

CSS Grid permite crear layouts en filas y columnas al mismo tiempo, perfecto para estructuras de página completas.

grid.css
/* Contenedor grid */
.grid {
  display: grid;

  /* 3 columnas iguales */
  grid-template-columns: repeat(3, 1fr);

  /* columnas de tamaño mixto */
  grid-template-columns: 200px 1fr 2fr;

  /* columnas que se ajustan automáticamente */
  grid-template-columns:
    repeat(auto-fill, minmax(240px, 1fr));

  /* 2 filas de altura fija + automática */
  grid-template-rows: 80px auto;

  gap: 20px;               /* espacio entre celdas */
}

/* Un elemento que abarca varias celdas */
.destacado {
  grid-column: 1 / 3;     /* de columna 1 a 3 */
  grid-row:    1 / 2;     /* de fila 1 a 2 */
}

/* Grid con áreas nombradas */
.pagina {
  display: grid;
  grid-template-areas:
    "header  header"
    "sidebar main  "
    "footer  footer";
}
header { grid-area: header; }
aside  { grid-area: sidebar; }
main   { grid-area: main; }

Capítulo 09

Diseño responsivo con Media Queries

Las media queries permiten aplicar estilos diferentes según el tamaño de la pantalla, haciendo que tu diseño funcione en celulares, tabletas y computadoras.

responsive.css
/* Estrategia Mobile First: primero el móvil */
.contenedor {
  display: grid;
  grid-template-columns: 1fr; /* 1 columna en móvil */
  gap: 16px;
}

/* Tableta: 768px o más */
@media (min-width: 768px) {
  .contenedor {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Escritorio: 1024px o más */
@media (min-width: 1024px) {
  .contenedor {
    grid-template-columns: repeat(3, 1fr);
  }
}

/* Pantalla grande: 1280px o más */
@media (min-width: 1280px) {
  .contenedor {
    grid-template-columns: repeat(4, 1fr);
  }
}

/* Solo si es pantalla táctil */
@media (hover: none) {
  .btn:hover { transform: none; }
}

/* Unidades relativas para texto responsivo */
h1 {
  font-size: clamp(1.5rem, 5vw, 3.5rem);
  /* mínimo 1.5rem, ideal 5% del viewport, máximo 3.5rem */
}
📱

Breakpoints comunes

480px móvil pequeño · 768px tableta · 1024px laptop · 1280px+ escritorio.

📐

Unidades relativas

rem relativo a la raíz · em relativo al padre · vw/vh relativo al viewport · % relativo al contenedor.

🔤

clamp()

Función CSS3 para tamaños fluidos con un mínimo y máximo: font-size: clamp(1rem, 3vw, 2rem).

📲

Mobile First

Diseña primero para pantallas pequeñas y añade complejidad con min-width. Es la práctica profesional estándar.

✦

Capítulo 10

Transiciones y animaciones

CSS3 permite crear movimiento y efectos visuales sin necesidad de JavaScript, usando transition para cambios suaves y @keyframes para animaciones completas.

animaciones.css
/* TRANSITION: suaviza un cambio de estado */
.boton {
  background:  #1572b6;
  padding:     10px 24px;
  border-radius: 8px;
  transition:  all 0.3s ease;
  /* propiedad  duración  curva */
}
.boton:hover {
  background:  #33a9dc;
  transform:   translateY(-4px);
  box-shadow:  0 8px 20px rgba(21,114,182,0.4);
}

/* TRANSFORM: transforma sin afectar el flujo */
.card:hover {
  transform: scale(1.04) rotate(1deg);
}

/* ANIMATION con @keyframes */
@keyframes latido {
  0%   { transform: scale(1);    opacity: 1;   }
  50%  { transform: scale(1.2);  opacity: 0.7; }
  100% { transform: scale(1);    opacity: 1;   }
}

.elemento {
  animation: latido 1.5s ease-in-out infinite;
  /* nombre  duración  curva  repetición */
}

Capítulo 11

Variables CSS (Custom Properties)

Las variables CSS3 permiten definir valores una sola vez y reutilizarlos en toda la hoja de estilos. Son el equivalente a las variables de programación dentro de CSS.

variables.css
/* Declarar variables en :root (disponibles globalmente) */
:root {
  /* Colores */
  --color-primario:  #1572b6;
  --color-acento:    #e91e8c;
  --color-fondo:     #f5f0e8;
  --color-texto:     #1a1a1a;

  /* Tipografía */
  --fuente-titulo:  'Playfair Display', serif;
  --fuente-cuerpo:  'Lato', sans-serif;
  --tamaño-base:    16px;

  /* Espaciados */
  --espacio-sm:  8px;
  --espacio-md:  16px;
  --espacio-lg:  32px;

  /* Bordes */
  --radio:       12px;
  --sombra:      0 4px 18px rgba(0,0,0,0.08);
}

/* Usar variables con var() */
.tarjeta {
  background:    var(--color-fondo);
  border-radius: var(--radio);
  padding:       var(--espacio-lg);
  box-shadow:    var(--sombra);
  font-family:   var(--fuente-cuerpo);
}

.titulo {
  color:       var(--color-primario);
  font-family: var(--fuente-titulo);
}

/* Cambiar tema oscuro con una sola regla */
@media (prefers-color-scheme: dark) {
  :root {
    --color-fondo: #1a1a2e;
    --color-texto: #ffffff;
  }
}
⭐

Punto clave – Variables CSS: Siempre define tu paleta de colores, fuentes y espaciados como variables en :root. Cuando necesites cambiar el color primario de todo tu sitio, solo tendrás que modificarlo en un solo lugar.

Capítulo 12

Tu ruta de aprendizaje CSS3

1

Escribe tu primer CSS externo

Crea un archivo estilos.css, vincúlalo a tu HTML con <link> y practica cambiando colores y fuentes.

2

Domina los selectores

Practica seleccionar elementos por etiqueta, clase e ID. Aprende las pseudo-clases :hover, :focus y :nth-child.

3

Comprende el modelo de caja

Experimenta con padding, margin y border. Agrega siempre box-sizing: border-box a todos tus proyectos.

4

Aprende Flexbox

Practica alineando elementos en fila y columna. Domina justify-content y align-items. Crea una barra de navegación y un grid de tarjetas.

5

Explora CSS Grid

Crea layouts de página completos con Grid: header, sidebar, main y footer usando grid-template-areas.

6

Hazlo responsivo

Agrega media queries a tus proyectos para que se vean bien en celular, tableta y escritorio. Practica la estrategia Mobile First.

7

Anima tu interfaz

Añade transiciones a botones y tarjetas. Crea una animación con @keyframes. Los detalles animados hacen la diferencia.

8

Usa variables CSS

Reorganiza tu proyecto usando variables en :root. Intenta crear un modo oscuro cambiando solo las variables.

🚀

Practica en línea: Usa codepen.io o css-tricks.com para experimentar con CSS en tiempo real sin instalar nada. El mejor aprendizaje es el de prueba y error.
