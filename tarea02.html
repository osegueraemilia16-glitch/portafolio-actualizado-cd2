Desarrollo Web

Lenguajes que dan vida a sitios y aplicaciones en el navegador

HTML 5
Lenguaje de Marcado

HTML (HyperText Markup Language) es el esqueleto de toda página web. Define la estructura y el contenido mediante etiquetas que el navegador interpreta.

index.html 
<!-- Estructura básica -->
<!DOCTYPE html>
<html lang="es">
  <head>
    <title>Mi página</title>
  </head>
  <body>
    <h1>Hola, mundo</h1>
    <p>Esto es HTML</p>
  </body>
</html>
CSS 3
Hoja de Estilos

CSS (Cascading Style Sheets) se encarga del diseño visual: colores, tipografías, márgenes, animaciones y la respuesta a diferentes tamaños de pantalla.

estilos.css 
/* Estilo de un botón */
.boton {
  background: #e91e8c;
  color: white;
  padding: 10px 20px;
  border-radius: 8px;
  transition: all 0.3s;
}
.boton:hover {
  transform: scale(1.05);
}
JavaScript
Lenguaje de Scripting

JavaScript da interactividad y lógica a las páginas web. Es el único lenguaje que corre directamente en el navegador, y hoy también en servidores gracias a Node.js.

app.js 
// Función de saludo
const saludar = (nombre) => {
  return `Hola, ${nombre}!`;
};

let usuario = "Emilia";
console.log(saludar(usuario));
// → Hola, Emilia!
PHP
Back-end Web

PHP es un lenguaje de servidor ampliamente usado para crear sitios dinámicos. Potencia plataformas como WordPress y Facebook en sus inicios.

index.php 
<?php
// Saludar al usuario
$nombre = "Emilia";

function saludar($n) {
  return "Hola, $n!";
}

echo saludar($nombre);
// → Hola, Emilia!
?>

Desarrollo de Escritorio

Lenguajes para crear aplicaciones que se instalan en computadoras

Python
Multipropósito · IA · Escritorio

Python es el lenguaje más popular del mundo hoy en día. Su sintaxis clara lo hace ideal para principiantes, IA, ciencia de datos y aplicaciones de escritorio.

main.py 
# Lista de materias
materias = [
  "Cultura Digital",
  "Matemáticas",
  "Inglés"
]

for m in materias:
  print(f"Materia: {m}")

# → Materia: Cultura Digital
Java
Multiplataforma · Empresarial

Java sigue el principio "escribe una vez, corre en cualquier lugar". Es base de Android y de grandes sistemas empresariales en bancos y corporaciones.

Main.java 
public class Main {
  public static void main(
      String[] args) {

    String nombre = "Emilia";
    System.out.println(
      "Hola, " + nombre
    );
    // → Hola, Emilia
  }
}
C#
Microsoft · Videojuegos · Windows

C# (C Sharp) es el lenguaje de Microsoft. Se usa para crear apps de Windows, videojuegos con Unity y servicios en la nube con .NET.

Program.cs 
using System;

class Program {
  static void Main() {
    string nombre = "Emilia";
    Console.WriteLine(
      $"Hola, {nombre}!"
    );
    // → Hola, Emilia!
  }
}
C++
Alto Rendimiento · Sistemas

C++ es el lenguaje de los motores de videojuegos, sistemas operativos y software de alto rendimiento. Es potente pero exige mayor precisión del programador.

main.cpp 
#include <iostream>
using namespace std;

int main() {
  string nombre = "Emilia";
  cout << "Hola, "
       << nombre << endl;
  // → Hola, Emilia
  return 0;
}

Diseño y Maquetación

Lenguajes y herramientas para crear interfaces, gráficos y documentos

HTML + CSS (maquetación)
Maquetación Web

Combinados, HTML y CSS son la herramienta de maquetación web por excelencia. Con Flexbox y Grid se crean layouts complejos y responsivos de forma elegante.

layout.css 
/* Layout con Grid */
.contenedor {
  display: grid;
  grid-template-columns:
    repeat(3, 1fr);
  gap: 20px;
}
.card {
  background: white;
  border-radius: 12px;
  padding: 16px;
}
SVG
Gráficos Vectoriales

SVG (Scalable Vector Graphics) es un lenguaje basado en XML para crear gráficos vectoriales escalables. Se usa en íconos, ilustraciones e infografías en la web.

grafico.svg 
<!-- Círculo y texto -->
<svg viewBox="0 0 100 100"
  xmlns="http://www.w3.org/2000/svg">

  <circle cx="50" cy="45"
    r="30" fill="#e91e8c"/>

  <text x="50" y="85"
    text-anchor="middle"
    fill="#880e4f">
    Diseño
  </text>
</svg>
Markdown
Documentación · Texto

Markdown es un lenguaje de marcado ligero para escribir documentos con formato usando solo texto plano. Se usa en README de GitHub, blogs y documentación técnica.

Reflexión final

¿Cuál debo aprender?

No existe un único lenguaje "mejor". La elección depende del objetivo: si quieres crear sitios web, empieza con HTML, CSS y JavaScript. Si te interesa la Inteligencia Artificial, Python es el camino. Si sueñas con videojuegos, explora C++ o C#.

En el contexto de Cultura Digital, entender la diversidad de lenguajes nos da una visión más completa del ecosistema tecnológico y nos prepara para ser ciudadanos digitales más críticos y creativos.
