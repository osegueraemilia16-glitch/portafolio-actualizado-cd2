Capítulo 01

¿Qué es HTML5?

HTML significa HyperText Markup Language (Lenguaje de Marcado de HiperTexto). No es un lenguaje de programación — es un lenguaje de estructura y contenido. Le dice al navegador qué es cada cosa: un título, un párrafo, una imagen, un enlace.

HTML5 es la quinta y actual versión del estándar, lanzada oficialmente en 2014 por el W3C (World Wide Web Consortium). Trajo consigo etiquetas semánticas, soporte nativo para audio y video, canvas para gráficos, APIs para geolocalización y mucho más.

🏗️

Estructura

HTML define la arquitectura de una página: qué va dónde y qué tipo de contenido es cada elemento.

🌐

Universal

Todo navegador del mundo (Chrome, Firefox, Safari) entiende HTML. Es el único lenguaje 100% universal de la web.

🆓

Libre y abierto

No necesitas instalar nada especial. Un bloc de notas y un navegador son suficientes para empezar.

🤝

Trabaja con CSS y JS

HTML estructura, CSS da estilo y JavaScript añade interactividad. Los tres forman el trío esencial de la web.

💡

Punto clave: HTML no "programa" — describe. Es como el plano de una casa: define cuántas habitaciones hay y dónde están, pero no decide de qué color pintarlas (eso es CSS).

Capítulo 02

Estructura básica de un documento HTML5

Todo documento HTML5 sigue una estructura fija que el navegador espera encontrar. Aprender esta plantilla de memoria es el primer paso fundamental.

plantilla-base.html 
<!-- 1. Declaración del tipo de documento -->
<!DOCTYPE html>

<!-- 2. Elemento raíz -->
<html lang="es">

  <!-- 3. Cabecera (no visible en pantalla) -->
  <head>
    <meta charset="UTF-8"/>
    <meta name="viewport"
          content="width=device-width, initial-scale=1.0"/>
    <title>Título de la pestaña</title>
    <link rel="stylesheet" href="estilos.css"/>
  </head>

  <!-- 4. Cuerpo (todo lo visible en pantalla) -->
  <body>
    <h1>Mi primer título</h1>
    <p>Mi primer párrafo.</p>
  </body>

</html>
📋

<!DOCTYPE html>

Le dice al navegador que este documento usa HTML5. Siempre va en la primera línea, sin excepción.

🌍

<html lang="es">

El elemento raíz que contiene todo. El atributo lang indica el idioma para accesibilidad y SEO.

🧠

<head>

Metainformación invisible: charset, viewport, título de pestaña, enlaces a CSS y scripts.

👁️

<body>

Todo lo que el usuario ve en pantalla: texto, imágenes, botones, videos — va dentro del body.

Capítulo 04

Atributos: cómo personalizar las etiquetas

Los atributos son información adicional que se agrega dentro de la etiqueta de apertura. Su formato siempre es nombre="valor".

atributos.html 
<!-- id: identificador único del elemento -->
<div id="encabezado">Encabezado</div>

<!-- class: nombre de grupo para aplicar estilos -->
<p class="texto-rosa">Párrafo estilizado</p>

<!-- src y alt en imágenes -->
<img src="logo.png"
     alt="Logo de la empresa"
     width="200"
     height="100"/>

<!-- href, target y rel en enlaces -->
<a href="pagina2.html"
   target="_blank"
   rel="noopener">
  Ir a página 2
</a>

<!-- style: CSS directamente en la etiqueta -->
<p style="color: #e91e8c; font-size: 18px;">
  Texto rosa
</p>
AtributoSe usa enFunción
idCualquier etiquetaIdentificador único. Solo puede haber un id por nombre en toda la página.
classCualquier etiquetaClase CSS. Muchos elementos pueden compartir la misma clase.
href<a>URL de destino del enlace.
src<img>, <script>, <video>Ruta al archivo fuente.
alt<img>Texto alternativo si la imagen no carga. Obligatorio para accesibilidad.
target<a>_blank (nueva pestaña) o _self (misma pestaña).
styleCualquier etiquetaCSS en línea (solo para casos puntuales).
type<input>, <button>Define el tipo de campo o botón.
placeholder<input>Texto de ayuda dentro del campo.
disabled<input>, <button>Desactiva el elemento.

Capítulo 05

Etiquetas semánticas: la gran novedad de HTML5

Antes de HTML5, todo se hacía con <div>. HTML5 introdujo etiquetas que describen su propio propósito, mejorando la accesibilidad, el SEO y la legibilidad del código.

estructura-semantica.html 
<body>

  <!-- Encabezado del sitio -->
  <header>
    <h1>Mi Blog</h1>
    <nav>
      <a href="#">Inicio</a>
      <a href="#">Blog</a>
      <a href="#">Contacto</a>
    </nav>
  </header>

  <!-- Contenido principal -->
  <main>
    <article>
      <h2>Mi primer artículo</h2>
      <p>Contenido del artículo...</p>
    </article>

    <aside>
      <h3>Artículos relacionados</h3>
    </aside>
  </main>

  <!-- Pie de página -->
  <footer>
    <p>© 2025 Mi Blog</p>
  </footer>

</body>
EtiquetaSignificadoUso típico
<header>EncabezadoLogo, nombre del sitio, menú de navegación
<nav>NavegaciónMenús de enlaces principal o secundario
<main>Contenido principalEl contenido único e importante de la página
<article>Artículo independientePost de blog, noticia, producto
<section>Sección temáticaGrupos de contenido relacionado
<aside>Contenido lateralPublicidad, info relacionada, barra lateral
<footer>Pie de páginaDerechos, contacto, enlaces secundarios
<figure>Figura con leyendaImagen con su descripción <figcaption>
<details>Contenido colapsableFAQs, acordeones simples
<mark>Texto resaltadoPalabras clave destacadas
🎯

Punto clave de HTML5: Usa etiquetas semánticas siempre que puedas en lugar de <div>. Los motores de búsqueda como Google leen estas etiquetas para entender tu página y posicionarla mejor.

Capítulo 06

Formularios y campos de entrada

Los formularios permiten recopilar información del usuario. HTML5 incorporó nuevos tipos de campos que facilitan la validación automática sin necesidad de JavaScript.

formulario-contacto.html 
<form action="enviar.php" method="POST">

  <!-- Campo de texto -->
  <label for="nombre">Nombre:</label>
  <input type="text"
         id="nombre"
         name="nombre"
         placeholder="Tu nombre"
         required/>

  <!-- Email con validación automática -->
  <label for="email">Correo:</label>
  <input type="email"
         id="email"
         placeholder="correo@ejemplo.com"/>

  <!-- Área de texto largo -->
  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje"
             rows="4"
             cols="40"></textarea>

  <!-- Selector desplegable -->
  <select name="materia">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
  </select>

  <!-- Botón de envío -->
  <button type="submit">Enviar</button>

</form>

Capítulo 07

Multimedia: audio, video e iframe

Una de las mayores novedades de HTML5 fue el soporte nativo para audio y video, eliminando la necesidad de plugins como Flash.

multimedia.html 
<!-- VIDEO nativo HTML5 -->
<video width="640" height="360"
       controls autoplay muted>
  <source src="video.mp4"
          type="video/mp4"/>
  Tu navegador no soporta video.
</video>

<!-- AUDIO nativo HTML5 -->
<audio controls>
  <source src="musica.mp3"
          type="audio/mpeg"/>
</audio>

<!-- IFRAME: incrustar YouTube -->
<iframe
  width="560" height="315"
  src="https://www.youtube.com/embed/ID"
  title="Video YouTube"
  allowfullscreen>
</iframe>

<!-- CANVAS: área de dibujo con JS -->
<canvas id="miCanvas"
        width="400"
        height="200">
</canvas>
🎬

<video>

Reproduce videos sin plugins. Soporta MP4, WebM y Ogg. El atributo controls muestra los controles de reproducción.

🎵

<audio>

Reproduce audio directamente. Formatos: MP3, WAV, OGG. Sin plugins externos.

🖼️

<iframe>

Incrusta otro documento o página web dentro de la tuya. Ideal para mapas, videos de YouTube y formularios externos.

🎨

<canvas>

Área de dibujo 2D controlada con JavaScript. Se usa para juegos, gráficos y animaciones interactivas.

Capítulo 08

Clases, IDs y la conexión con CSS

Clases e IDs son el puente entre HTML y CSS. Permiten aplicar estilos específicos a elementos concretos.

clases-ids.html + estilos.css 
<!-- HTML -->
<div id="encabezado">
  <h1 class="titulo-principal">
    Cultura Digital 2
  </h1>
</div>

<p class="texto-rosa">
  Primer párrafo
</p>
<p class="texto-rosa">
  Segundo párrafo (misma clase)
</p>

/* CSS */
/* Seleccionar por ID con # */
#encabezado {
  background: #fce4ec;
  padding: 20px;
}

/* Seleccionar por clase con . */
.texto-rosa {
  color: #e91e8c;
  font-size: 1.1rem;
}
#️⃣

ID ( id="" )

Único por página. En CSS se selecciona con #nombre. Úsalo para secciones únicas: encabezado, pie, menú principal.

🏷️

Clase ( class="" )

Se puede repetir en múltiples elementos. En CSS se selecciona con .nombre. Ideal para botones, tarjetas, párrafos.

🔢

Múltiples clases

Un elemento puede tener varias clases separadas por espacio: class="card activa destacada".

🔗

Anclas con ID

Los IDs también sirven para navegar dentro de la página: <a href="#seccion2"> lleva al elemento con ese ID.

Capítulo 09

Tu ruta de aprendizaje paso a paso

Sigue estos pasos en orden para aprender HTML5 de forma sólida y progresiva.

1

Instala un editor de código

Descarga Visual Studio Code (gratis). Instala la extensión "Live Server" para ver tus cambios en tiempo real en el navegador.

2

Domina la plantilla base

Memoriza la estructura básica: DOCTYPE, html, head, body. Crea tu primer archivo .html y ábrelo en el navegador.

3

Practica las etiquetas de texto

Crea páginas con h1-h6, párrafos, negritas, cursivas, listas y enlaces. Sin CSS todavía — solo estructura.

4

Agrega imágenes y enlaces

Practica con <img> y <a>. Aprende rutas relativas (imágenes en tu carpeta) y absolutas (URLs externas).

5

Aprende etiquetas semánticas

Reconstruye tus páginas usando header, main, section, article, footer en lugar de divs genéricos.

6

Crea formularios

Practica todos los tipos de input. Crea un formulario de contacto completo con validación HTML5 nativa.

7

Conecta con CSS

Agrega un archivo externo .css enlazado con <link>. Aquí tu HTML cobra vida visualmente.

8

Construye tu primer proyecto

Crea una página de portafolio personal con todo lo aprendido: header, nav, secciones, formulario de contacto y footer.

🚀

Recurso recomendado: La documentación oficial de HTML en MDN Web Docs (developer.mozilla.org) está disponible en español y es la referencia más completa y confiable para consultar cualquier etiqueta o atributo.
