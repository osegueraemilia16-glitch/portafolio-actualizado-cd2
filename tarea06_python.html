Capítulo 01

¿Qué es Python?

Python es un lenguaje de programación de propósito general, creado por Guido van Rossum en 1991. Su filosofía principal es la legibilidad del código: un programa en Python debe poder leerse casi como inglés natural. Es el lenguaje número 1 en Inteligencia Artificial, ciencia de datos y automatización.

📖

Fácil de leer

Su sintaxis limpia y sin llaves ni puntos y coma lo hace ideal para principiantes y expertos.

🌍

Multipropósito

Se usa en web, IA, ciencia de datos, automatización, videojuegos, ciberseguridad y más.

🐍

Interpretado

No necesita compilarse: el intérprete ejecuta el código línea por línea, facilitando la depuración.

📦

Gran ecosistema

Más de 400,000 librerías en PyPI: NumPy, Pandas, TensorFlow, Django, Flask y muchas más.

🤖

Rey de la IA

Casi todos los modelos de Inteligencia Artificial modernos, incluyendo ChatGPT, se desarrollan con Python.

🆓

Libre y gratuito

Python es de código abierto. Puedes descargarlo, usarlo y distribuirlo sin costo alguno.

💡

Punto clave: Python usa la indentación (sangría) para definir bloques de código, en lugar de llaves como Java o C++. Esto obliga a escribir código ordenado y legible desde el principio.

✦

Capítulo 02

Variables y tipos de datos

Una variable es un espacio en memoria donde guardamos información. En Python no necesitas declarar el tipo — Python lo detecta automáticamente.

variables.py
# ── Tipos de datos básicos ──────────────────

# Entero (int)
edad       = 17

# Decimal (float)
promedio   = 9.5

# Texto (str)
nombre     = "Emilia"

# Booleano (bool): solo True o False
activo     = True
reprobado  = False

# Ningún valor (NoneType)
resultado  = None

# Verificar el tipo de una variable
print(type(edad))       # <class 'int'>
print(type(promedio))   # <class 'float'>
print(type(nombre))     # <class 'str'>

# Conversión entre tipos (casting)
edad_texto = str(edad)        # 17 → "17"
num        = int("42")        # "42" → 42
decimal    = float("3.14")   # "3.14" → 3.14

Punto clave – Nombres de variables: Usa nombres descriptivos en minúsculas con guion bajo: nombre_alumna, promedio_final. Evita nombres como x, a1 o dato. Un buen nombre hace el código autoexplicativo.

Capítulo 03

Operadores

Los operadores permiten realizar cálculos, comparaciones y operaciones lógicas con los datos.

operadores.py
# ── Operadores Aritméticos ──────────────────
a, b = 10, 3

print(a + b)   # 13  → suma
print(a - b)   # 7   → resta
print(a * b)   # 30  → multiplicación
print(a / b)   # 3.33→ división
print(a // b)  # 3   → división entera
print(a % b)   # 1   → módulo (residuo)
print(a ** b)  # 1000→ potencia

# ── Operadores de Comparación ───────────────
print(10 == 10)  # True  → igual
print(10 != 5)   # True  → diferente
print(10 > 3)    # True  → mayor que
print(10 < 3)    # False → menor que
print(10 >= 10) # True  → mayor o igual

# ── Operadores Lógicos ──────────────────────
print(True and False)  # False → ambos verdaderos
print(True or  False)  # True  → al menos uno
print(not True)         # False → negación

Capítulo 04

Estructuras de control

Las estructuras de control permiten tomar decisiones y repetir acciones en nuestro programa.

estructuras_control.py
# ── Condicional if / elif / else ────────────
calificacion = 85

if calificacion >= 90:
    print("Excelente 🌟")
elif calificacion >= 70:
    print("Aprobado ✅")
else:
    print("Reprobado ❌")
# → Aprobado ✅

# ── Ciclo for (recorrer secuencias) ─────────
for i in range(1, 6):
    print(f"Tarea {i}")
# → Tarea 1, Tarea 2 ... Tarea 5

# ── Ciclo while (repetir mientras) ──────────
contador = 0
while contador < 3:
    print(f"Vuelta {contador}")
    contador += 1
# → Vuelta 0, Vuelta 1, Vuelta 2

# ── break y continue ────────────────────────
for n in range(10):
    if n == 5:
        break       # detiene el ciclo
    if n % 2 == 0:
        continue    # salta a la siguiente iteración
    print(n)       # imprime 1, 3
🔀

if / elif / else

Ejecuta bloques distintos según condiciones. Pueden encadenarse tantos elif como sean necesarios.

🔁

for

Recorre una secuencia (lista, rango, cadena). Ideal cuando conoces cuántas veces repetir.

🔄

while

Repite mientras una condición sea verdadera. Útil cuando no sabes cuántas veces se repetirá.

⛔

break / continue

break detiene el ciclo completamente. continue salta al siguiente ciclo sin ejecutar el resto.

Capítulo 05

Listas, tuplas, diccionarios y conjuntos

Las colecciones permiten guardar múltiples valores en una sola variable. Son fundamentales en cualquier programa Python.

colecciones.py
# ── LISTA: ordenada, modificable ────────────
materias = ["Matemáticas", "Python", "Inglés"]
materias.append("Historia")   # agregar al final
materias.remove("Inglés")    # eliminar elemento
print(materias[0])           # → Matemáticas
print(len(materias))         # → 3

# ── TUPLA: ordenada, INMUTABLE ──────────────
colores = ("rosa", "azul", "blanco")
print(colores[1])            # → azul
# colores[0] = "rojo"  → ERROR, no modificable

# ── DICCIONARIO: clave-valor ─────────────────
alumna = {
    "nombre":  "Emilia",
    "edad":    17,
    "escuela": "CBTis 144"
}
print(alumna["nombre"])      # → Emilia
alumna["promedio"] = 9.8    # agregar clave

# ── CONJUNTO: sin duplicados ─────────────────
numeros = {1, 2, 3, 2, 1}
print(numeros)              # → {1, 2, 3}

Capítulo 06

Funciones

Una función es un bloque de código reutilizable que realiza una tarea específica. Son el pilar de la programación organizada.

funciones.py
# ── Función básica ──────────────────────────
def saludar():
    print("¡Hola, mundo!")

saludar()  # → ¡Hola, mundo!

# ── Con parámetros ──────────────────────────
def saludar(nombre, edad):
    print(f"Hola {nombre}, tienes {edad} años")

saludar("Emilia", 17)

# ── Con valor por defecto ───────────────────
def potencia(base, exp=2):
    return base ** exp

print(potencia(3))      # → 9  (exp=2 por defecto)
print(potencia(2, 10)) # → 1024

# ── Función lambda (anónima) ─────────────────
cuadrado = lambda x: x ** 2
print(cuadrado(5))      # → 25

# ── Retorno múltiple ─────────────────────────
def min_max(lista):
    return min(lista), max(lista)

minimo, maximo = min_max([3, 1, 7, 2])
print(minimo, maximo)   # → 1 7
⭐

Punto clave – DRY: El principio Don't Repeat Yourself (No te repitas) dice que si escribes el mismo código más de dos veces, conviértelo en función. Esto hace tu código más limpio, fácil de mantener y menos propenso a errores.

✦

Capítulo 07

Programación Orientada a Objetos (POO)

La POO es un paradigma que organiza el código en clases (plantillas) y objetos (instancias). Python es un lenguaje orientado a objetos desde su núcleo.

poo.py
# ── Definir una Clase ────────────────────────
class Alumna:

    # Constructor: se ejecuta al crear el objeto
    def __init__(self, nombre, grado):
        self.nombre = nombre
        self.grado  = grado
        self.calificaciones = []

    # Método: función dentro de la clase
    def agregar_calificacion(self, nota):
        self.calificaciones.append(nota)

    def promedio(self):
        if self.calificaciones:
            return sum(self.calificaciones) / len(self.calificaciones)
        return 0

    def presentarse(self):
        return f"Soy {self.nombre} de {self.grado}"

# ── Crear un objeto (instancia) ──────────────
emilia = Alumna("Emilia", "6° semestre")
emilia.agregar_calificacion(9.5)
emilia.agregar_calificacion(9.8)

print(emilia.presentarse())  # → Soy Emilia de 6° semestre
print(emilia.promedio())     # → 9.65
🏛️

Clase

Plantilla o molde que define atributos (datos) y métodos (comportamientos) de un tipo de objeto.

📦

Objeto

Instancia concreta de una clase. Puedes crear muchos objetos a partir de una sola clase.

🔧

__init__

Método constructor que se ejecuta automáticamente al crear el objeto. Inicializa sus atributos.

🧬

Herencia

Una clase puede heredar atributos y métodos de otra clase padre, evitando duplicar código.

Capítulo 08

Módulos y librerías esenciales

Un módulo es un archivo Python con funciones y variables listas para usar. Python incluye cientos de módulos estándar y millones de librerías externas.

modulos.py
# ── Importar módulos estándar ────────────────
import math
print(math.sqrt(16))     # → 4.0  (raíz cuadrada)
print(math.pi)           # → 3.14159...

import random
print(random.randint(1, 10)) # número aleatorio 1-10
print(random.choice(["a", "b", "c"]))

import datetime
hoy = datetime.date.today()
print(hoy)                # → 2025-05-12

# ── Importar solo lo necesario ───────────────
from math import sqrt, pi
print(sqrt(25))           # → 5.0 (sin prefijo math.)

# ── Instalar librerías externas (en terminal) ─
# pip install numpy pandas matplotlib

# ── NumPy: cálculo numérico ──────────────────
import numpy as np
arr = np.array([1, 2, 3, 4])
print(arr * 2)            # → [2, 4, 6, 8]

Capítulo 09

Tu ruta para aprender Python

1

Instala Python y un editor

Descarga Python desde python.org. Instala Visual Studio Code con la extensión de Python. Escribe tu primer print("Hola mundo").

2

Domina variables y tipos de datos

Practica creando variables de diferentes tipos. Experimenta con conversiones entre int, float y str.

3

Aprende operadores y condiciones

Crea programas que tomen decisiones con if/elif/else. Practica los operadores de comparación y lógicos.

4

Practica ciclos

Crea listas y recórrelas con for. Usa while para repetir hasta que se cumpla una condición.

5

Crea tus propias funciones

Convierte cada tarea repetitiva en función. Practica el retorno de valores y los parámetros por defecto.

6

Explora POO

Crea una clase con atributos y métodos. Instancia objetos y experimenta con la herencia.

7

Usa librerías

Instala NumPy y Pandas con pip. Crea tu primera gráfica con Matplotlib. Explora el ecosistema.

8

Construye un proyecto

Crea un programa completo: una calculadora, un juego de adivinanza, o un analizador de datos con CSV.
